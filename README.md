# Velocibet

이 프로젝트는 사용자 간 웹 기반 실시간 메세지 송수신을 지원하는 개인 프로젝트입니다.
(서비스 URL: https://www.velocibet.com/)

## 주요 기능

- 실시간 메시지 송수신
- 사용자 인증 (로그인 / 회원가입)
- 메시지 창 내 파일 전송

## 개발 중인 기능
- Cloudflare R2 기반 파일 저장

## 기술 스택

### Frontend
- Nuxt (Vue)
- TypeScript

### Backend
- NestJS
- PostgreSQL

### Storage
- Cloudflare R2

## 기타
이 프로젝트는 배포용으로 제작하지 않았으므로 별도의 설치 안내는 제공하지 않습니다.
